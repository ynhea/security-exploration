## 🎯 Project Goal

이 프로젝트의 목표는 단순한 공부가 아니라 **실제 보안 직무를 경험**하는 것입니다.

- 🔍 정보보안 직무 체험

- 📈 실무 중심 프로젝트 수행
- 🧭 나에게 맞는 분야 찾기
- 📚 GitHub 포트폴리오 완성
